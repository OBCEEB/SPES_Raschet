# Changelog

## 0.1.0 - 2026-04-06

### Added

- базовая документация проекта (`README`, `docs/*`);
- правила версионирования (`docs/VERSIONING.md`);
- копирование `csv/json` в output при сборке.

### Changed

- стабильная загрузка файлов данных с fallback-поиском путей;
- fallback по кодировке CSV (UTF-8 -> Windows-1251);
- улучшены сообщения об ошибках загрузки данных в UI;
- очищены и нормализованы комментарии по коду.

### Fixed

- устранена проблема сопоставления регионов из-за некорректной декодировки CSV;
- сборка приведена к чистому состоянию (без warning/error).
