# Troubleshooting

## При клике по региону: "нет данных для этого региона"

Проверить:

- корректно ли загрузился `settlements.csv`;
- читается ли файл в правильной кодировке;
- существует ли регион в `regions_bounds.json`.

## Пустые таблицы и графики

Проверить наличие файлов:

- `DailyTotalData.csv`
- `IrradianceData.csv`
- `SunPosition_Altitude.csv`
- `SunPosition_Azimuth.csv`

Также убедиться, что файлы попали в output после сборки.

## Ошибка чтения файлов при запуске

- проверить, что данные лежат рядом с `.exe` или в корне проекта;
- выполнить `dotnet build` заново;
- убедиться, что нет блокировки файлов антивирусом или правами доступа.

## Сборка предупреждает о таргет-фреймворке

Для базовой версии проекта предупреждение EOL подавлено флагом:

`CheckEolTargetFramework=false`.

При следующем этапе развития проекта рекомендуется перейти на поддерживаемый LTS-фреймворк.
